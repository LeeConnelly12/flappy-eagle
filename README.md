# Flappy Eagle

## About

Flappy bird clone, except you're a patriotic american bald eagle with a gun.

## Install PNPM

Through NPM

```
npm install -g pnpm
```

Mac Homebrew

```
brew install pnpm
```

## Building

Install dependencies

```
pnpm i
```

Run dev server

```
pnpm dev
```

Build the assets

```
pnpm build
```
